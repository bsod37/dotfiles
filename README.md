## Description
This is my dotfiles from laptop with Manjaro i3.

## Dependencies
To run all this configuration files you need the following:
* i3-gaps
* i3blocks
* compton
* urxvt
* dmenu
* dunst
* nitrogen
* mocp
* Vertex-Maia-Dark
* Vibrancy-Colors-Full-Dark

Fonts:
* Roboto
* Terminus
